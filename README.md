# Symfony2 ベースのサンプル

このサンプルはドメイン駆動設計 (DDD: Domain-Driven Design) に基づいた、PHP アプリケーションフレームワーク [Symfony](http://symfony.com/) ベースのアプリケーションです。

以下の DDD パターンを使っています。

 * Specification（仕様）
 * Repository（リポジトリ）
 * Entity（エンティティ）

また、アプリケーションレイヤーにてコントローラーからビューモデルを分離する例にもなっています。
